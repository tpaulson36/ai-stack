# End-of-Session Compress

Write a concise session log capturing what happened this session.

1. Determine today's date and a 2-4 word topic slug (e.g. `2026-04-08-make-automation`)
2. Summarize what was accomplished (3-5 bullet points, outcomes not actions)
3. List all files modified
4. List open items / next steps with checkboxes
5. Save the log to:
   - Project root `Session Logs/YYYY-MM-DD-<topic>.md` if that folder exists
   - `~/Documents/TysThoughts/Claude/Session Logs/YYYY-MM-DD-<topic>.md` always

Format:
```
# YYYY-MM-DD — Topic

**Project:** <project name>
**Session Goal:** <one line>

## Accomplished
- ...

## Files Modified
- ...

## Open Items
- [ ] ...
```
